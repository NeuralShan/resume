\begin{joblong}{Data \& AI Engineer, Accenture}{Sep 2024 - present}

\item \textbf{Agentic Data Quality Platform (Internal):} Built an enterprise data quality application that ingests table metadata and schema information from Google BigQuery and routes it to specialized LLM agents (anomaly detection, null checks, format and missing-value validation), each autonomously generating targeted SQL validation queries.

\item Implemented a SQL safety layer that validates generated queries before execution, blocking DML and malformed statements; persisted queries and result JSON to GCS buckets and surfaced findings through an interactive Angular dashboard with chart-based visualizations.

\item \textbf{Agentic Email Automation \& Response Generation (Client):} Designed an end-to-end pipeline that monitors an inbox and triggers on email arrival, ingesting email content as JSON into Azure Cosmos DB and attachments into Azure Blob Storage.

\item Developed a preprocessing agent that distills emails into clean, agent-ready JSON and extracts attachment content (OCR from images, text from PDF/Excel) via an LLM-based extraction tool, enriching the payload for downstream agents.

\item Built a classification agent that routes emails into business use cases, with dedicated use-case agents performing actions such as SharePoint storage, instant replies, and draft generation; persisted processed results to Cosmos DB.

\item Exposed results through a FastAPI service powering a React frontend with multiple dashboards: side-by-side original and drafted email (with direct web links), archived emails, email/category analytics, and an agentic token-usage telemetry dashboard.

\end{joblong}
